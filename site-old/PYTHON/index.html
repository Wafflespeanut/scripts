Basic data structures

(lists & sets)


							string = '''Hello, world!'''		# multi-line string literal
							list_ = list(string)		# print list_
							set_ = set(string)			# print set_

lazy lists...


							print list_[:4]		# which is just list_[0:4]
							print list_.index('w')		# find something
							assert ''.join(list_[2::2]) == string[2::2]		# slice & dice!

packed sets...


							set_[2]		# impossible!
							set_.update('Hello'.split())		# you can also pass a list

The `is` vs `==` problem...


							string = '''Hello, world!'''
							list_ = list(string)		# print list_
							set_ = set(string)			# print set_

							set(list(string)) == set(string)	# True
							set(list(string)) is set(string)	# False?

							[] == [] and [] is []		# False?
							256 is 32 * 8				# (small integers) True
							65536 is 256 * 256			# (big ones) False?

							"zzz" is "z" * 3
							x = "z"				# a binding changes the result
							"zzz" is x * 3		# Meh...

Basic data structures

Hashmaps are awesome!


							dict_ = dict(zip(range(1, 10), range(1, 10)))
							dict_ = dict(zip(enumerate(range(1, 10))))

							dict_ = dict((i, i * i) for i in range(10))
							dict_ = { i: i * i for i in range(10) }

							dict_[0] = 'Meow...'		# reason for immutability
							dict[0.111] = 'Grrr!!!'
							dict_['dog'] = 'Woof!'			# apparently, we can do anything!
							dict_[tuple(range(3))] = True
							dict_[range(3)] = False			# well, almost...

							print "{} {}".format(dict_[0], dict_['dog'])
